# 🚀 AI To-Do Assistant

## 📖 Overview

An intelligent To-Do application that leverages AI and natural language processing to help users organize, prioritize, and interact with tasks more efficiently through a chat-based interface.

## ✨ Features

* 📝 Create, update, and delete tasks in real-time
* 🤖 AI-powered task suggestions using local LLMs
* 💬 Chat interface for managing tasks conversationally
* 📅 Smart task handling and organization

## 🛠️ Tech Stack

* **Python**
* **Streamlit** (for interactive UI)
* **NLP / Machine Learning**
* **Ollama + Mistral 7B** (local LLM integration)
